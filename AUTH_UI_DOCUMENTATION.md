## üîÑ User Flow

### New Company Registration:
1. Navigate to Sign Up page
2. Fill in company details:
   - Company Name
   - Admin Name
   - Email
   - Phone
   - Password
   - Confirm Password
3. Optionally upload company logo
4. Read HR/Admin notice
5. Click "Sign Up"
6. System creates company + HR account
7. Auto-redirects to Admin Dashboard

### Employee Login:
1. Navigate to Sign In page
2. Read "Employee Login" information
3. Enter Login ID (from HR)
4. Enter password (from HR)
5. Click "Sign In"
6. Redirects to Employee Dashboard

### HR/Admin Login:
1. Navigate to Sign In page
2. Enter email address
3. Enter password
4. Click "Sign In"
5. Redirects to Admin Dashboard

## üîê Security Notes

### Login ID System:
- **Auto-Generated** by system when HR creates employee
- **Format**: OIJODO202420001
  - OI = Company Initials (Oxford Industries)
  - JO = First 2 letters of first name (John)
  - DO = First 2 letters of last name (Doe)
  - 2024 = Year of joining
  - 0001 = Serial number (employee number for that year)

### Password System:
- **HR/Admin**: Set their own password during registration
- **Employees**: System generates initial password
- **First Login**: Employees should change password
- **Security**: All passwords hashed with bcrypt

### Access Control:
- **Sign Up**: HR/Admin only
- **Employee Creation**: HR/Admin only (from dashboard)
- **Employee Login**: Use auto-generated Login ID
- **HR Login**: Use email address

This creates a secure, user-friendly authentication system with clear guidance for all user types!
